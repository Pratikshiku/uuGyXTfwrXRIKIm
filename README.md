## Recruitment System Multi-role

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/3b56e345a2244d89b881c2574f5351a5/ghb20250917) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript`, etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can be used to run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): regular users, enterprises, administrators.

**Regular User Functions:**
1. Login
2. Register
3. Homepage
4. Enterprises
5. Recruitment Information
6. Recruitment News
7. Message Feedback
8. Personal Center
9. My Collection



**Enterprise Functions:**
1. Login
2. Register
3. Change Password
4. Personal Information
5. Position Type Management
6. Recruitment Information Management
7. Application Record Management







**Administrator Functions:**
1. Login
2. Personal Center (change password, personal information)
3. Enterprise Management
4. User Management
5. Position Type Management
6. Recruitment Information Management
7. Application Record Management
8. Message Feedback
9. System Management (recruitment news, carousel image management)
